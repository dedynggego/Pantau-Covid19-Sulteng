ABOUT



Aplikasi pantau Covid-19 Sulawesi Tengah ini dibuat dengan menggunakan python dan flask di back-end, serta bootstrap dan chart.js di front-end. Data yang disajikan di-consume dari beberapa API yang tersedia secara publik dan legal.

Beberapa url API yang digunakan sebagai berikut:
https://apicovid19indonesia-v2.vercel.app/api/indonesia/more
https://apicovid19indonesia-v2.vercel.app/api/indonesia/provinsi/more
https://apicovid19indonesia-v2.vercel.app/api/indonesia/harian
https://data.covid19.go.id/public/api/prov.json
